# GDR-L2-003 — Annulation de GDR-L2-002

**Date** : 2026-06-08
**Statut** : Accepté
**Annule** : ADR-L2-002 — Un fichier par feature

## Contexte

ADR-L2-002 a été créé par erreur dans le dossier `adr-l2/`, qui gouverne les épics. La règle qu'il contient — un fichier par feature — concerne la structure des features et appartient au niveau L3.

## Décision

ADR-L2-002 est annulé. La règle est redéfinie correctement dans `adr-l3/ADR-L3-001-un-fichier-par-feature.md`.
