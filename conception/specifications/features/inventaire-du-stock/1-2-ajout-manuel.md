# Feature 1.2 — Ajout manuel

**Description** : L'utilisateur ajoute un médicament manuellement (nom, quantité, date de péremption), pour les boîtes dont le code-barres est illisible ou absent.
**Critère d'acceptation** : Un médicament ajouté manuellement apparaît dans le stock. La saisie complète prend moins de 30 secondes.
