# Feature 3.1 — Configuration profil remboursement

**Description** : L'utilisateur saisit une fois son taux de remboursement mutuelle dans ses paramètres. Ce taux est appliqué à tous les calculs de reste à charge.
**Critère d'acceptation** : L'utilisateur peut saisir et modifier son taux mutuelle en moins de 60 secondes. Le taux est persisté localement.
